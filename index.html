Play-siena

Siena Module for Play! Framework

Download this project as a .zip file Download this project as a tar.gz file

play-siena module for q42 fork

Created because of a jsonserialization bug concerning Boolean and boolean fields.

Download versions:

  • play-siena-2.0.7.1.q42.zip

Usage in dependencies.yml

require:
    - play
    - q42siena -> play-siena 2.0.7.1.q42



repositories:
    - q42siena:
        type:        http
        artifact:    "http://q42.github.io/play-siena/[module]-[revision].zip"
        contains:
            - q42siena -> *
